Usage Instructions

Click any skill card to open the Technical Specification panel. The panel slides in from the right with a spring animation featuring:

• NothingDot font for the skill title
• Red/Cyan gradient mastery bar with animated fill
• Detail grid showing sync status, domain, and priority
• Sub-skills matrix displaying related technologies
• Initialize Project button with glitch effect on hover
• Brutalist edges using clip-path
• Blurred glass background with backdrop-filter

Import: import { TechSpecCard } from './TechSpecCard.js';
Initialize: const specCard = new TechSpecCard();
Show: specCard.show(skillData);
